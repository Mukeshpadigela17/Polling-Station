# Polling-Station

🗳️ Mini Voting System in C

This is a simple console-based voting system built using the C programming language. The project demonstrates basic programming concepts such as functions, arrays, control structures, and user input handling.

🚀 Features

- Cast vote for one of the predefined candidates
- View vote counts (including spoiled votes)
- View the current leading candidate
- Menu-driven interface for ease of use

🛠️ Technologies Used

- Programming Language: **C**
- Compiler: GCC or any C compiler
- Platform: Console/Terminal

👨‍💻 Usage
Choose options from the menu:

1 to cast your vote.

2 to view current vote counts.

3 to see the leading candidate.

0 to exit the program.

👥 Candidates
Alice

Bob

Charlie

Votes outside the valid range are recorded as spoiled votes.

📚 Learning Objectives
This project helps to understand:

Basic structure of a C program

Functions and modular coding

Array-based data handling

Input validation

Simple menu-driven logic
